# Project

## Task 1: Pizza Price Calculator (4-for-3 Offer)

This program simulates a 4-for-3 pizza offer from Beckett Pizza Plaza. The user is asked to enter the prices of four pizzas. The program validates the input to ensure only positive numbers are accepted.The cheapest pizza is automatically made free, and the total amount to pay is calculated based on the remaining three pizzas.

Concepts Used

- While loops
- Input validation with try and except
- Conditional statements
- Basic arithmetic operations

## Task 2: Password Checker

This program checks the strength of a password and performs multiple security checks. First, it verifies that the password is at least 9 characters long. Then, it randomly asks the user to confirm characters from specific positions in the password.

Concepts Used
- String handling
- Length checking
- Random number generation
- Conditional statements
- Program termination using exit()
