# Notebooks

## Weekly Assignments
This repository contains weekly Python programming assignments from Week 2 to Week 8 for the Fundamentals of Computer Programming module.

### Week 2 – Basic Programming Concepts
Topics Covered:

User input using input()
Output formatting with f-strings
Variables and basic data types
Arithmetic operations
Integer division (//) and modulus (%)
Simple conditional expressions

### Week 3 – Control Flow and Loops
Topics Covered:

Conditional statements (if, else)
String handling and input validation
Conditional (ternary) expressions
for and while loops
Repetition until valid input
Multiplication tables

### Week 4 – Functions and Data Processing
Topics Covered:

Defining and calling functions
Parameters and return values
Boolean logic
String methods and slicing
Lists and list comprehensions
Built-in functions (max(), min())
Basic statistics

### Week 5 – Command-Line Programs and File Handling
Topics Covered:

Command-line arguments (sys.argv)
Operating system information
Exception handling (try / except)
File input and output
File backup creation

### Week 6 – Algorithms and Encryption
Topics Covered:

Binary number conversion
Factor and prime number calculations
Algorithm optimisation
String manipulation
Simple encryption and decryption
Random data generation

### Week 7 – Sets, Dictionaries and Data Analysis
Topics Covered:

Sets and set operations
Removing duplicate values
Dictionaries and key–value pairs
Case-insensitive input handling
Letter frequency analysis

### Week 8 – File Processing and Unix-Style Tools
Topics Covered:

File input/output
Line-by-line file processing
Command-line utilities
Counting lines and characters
Dictionary-based spell checking
